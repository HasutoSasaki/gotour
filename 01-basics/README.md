# 01. Basics

Go の基本的な構文について学習します。

## ディレクトリ一覧

- `hello/` - Hello, World!プログラム
- `sandbox/` - 乱数生成の例
- `packages/` - パッケージのインポートと使用

## 学習ポイント

- `package main`とは
- `import`文の使い方
- `fmt.Println()`の使い方
- 標準ライブラリの使用方法

## 実行方法

```bash
cd hello && go run main.go
cd sandbox && go run main.go
cd packages && go run main.go
```
